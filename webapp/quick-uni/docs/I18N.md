# QuickUni - Hướng dẫn Đa ngôn ngữ (i18n)

Dự án sử dụng `next-intl` để hỗ trợ đa ngôn ngữ.

## 1. Cấu trúc thư mục i18n
- `messages/`: Chứa các tệp JSON định nghĩa bản dịch (ví dụ: `vi.json`, `en.json`).
- `src/i18n/`: Chứa cấu hình routing và request cho i18n.
- `src/app/[locale]/`: Toàn bộ cấu trúc app nằm trong folder động `[locale]`.

## 2. Cách sử dụng

### 2.1. Trong Server Components
```tsx
import { useTranslations } from 'next-intl';

export default function MyComponent() {
  const t = useTranslations('Namespace');
  return <h1>{t('title')}</h1>;
}
```

### 2.2. Trong Client Components
Tương tự như Server Components, nhưng cần đảm bảo component được bọc trong một Provider (thường đã được cấu hình sẵn trong layout).

### 2.3. Trong Server Actions
Sử dụng `getTranslations` từ `next-intl/server`.

## 3. Thêm ngôn ngữ mới
1. Tạo tệp JSON mới trong `messages/`.
2. Cập nhật danh sách locales trong `src/i18n/routing.ts`.
