Dự án QuickUni - Tổng quan hệ thống

Phân tích cấu trúc và các thành phần chính của hệ thống quản lý đại học

Kiến trúc kỹ thuật (Tech Stack)
  • Frontend: Next.js 15 (App Router), TypeScript, Tailwind CSS, Shadcn UI
  • Backend: Next.js Server Actions, NextAuth.js v4
  • Database: PostgreSQL, Drizzle ORM
  • Internationalization: next-intl (Tiếng Anh & Tiếng Việt)

Các phân hệ cốt lõi đã xác định:

Auth & Security

RBAC (Role Based Access Control), Audit Log cho tài khoản.

User & Profile

Hồ sơ động (Dynamic Profiles) với Profile Schema và Profile Fields.

Academic

Quản lý Học kỳ, Khoa, Chuyên ngành, Chương trình đào tạo (Curriculum), Môn học.
